# 🍎 Apple Store UI Clone (GSAP + Three.js)

A modern Apple Store-inspired UI clone built to learn and experiment with **GSAP animations** and **Three.js 3D rendering**.

## 🚀 Overview

This project replicates the look and feel of the Apple Store website while focusing on:

- Smooth, scroll-based animations using **GSAP**
- Interactive 3D iPhone model using **Three.js**
- Responsive UI built with **Next.js** and **Tailwind CSS**

---

## ✨ Features

- 🎯 Apple-style clean and minimal UI
- 🎬 Smooth animations powered by GSAP
- 📱 Interactive 3D iPhone model:
  - Rotate
  - Spin
  - User-controlled interaction
- ⚡ Fast performance with Next.js
- 🎨 Utility-first styling using Tailwind CSS
- 🌐 Fully deployed on Vercel

---

## 🛠️ Tech Stack

- **Framework:** Next.js  
- **Styling:** Tailwind CSS  
- **Animations:** GSAP  
- **3D Rendering:** Three.js (React Three Fiber / Drei)  
- **Deployment:** Vercel
